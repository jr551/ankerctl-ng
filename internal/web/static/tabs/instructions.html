Connecting OrcaSlicer

  1. Open OrcaSlicer and edit your printer.
  2. Set Host Type to OctoPrint.
  3. Set Hostname / URL to: {{.RequestHost}}:{{.RequestPort}}
  4. Set your API key only if you enabled write protection here.
  5. Use Send or Send and Print from OrcaSlicer.
  6. Use Send and Print when you want the job to begin immediately.
This experimental build is tuned for OrcaSlicer first. Keep the setup simple and use the OctoPrint upload path.
